## Le Joint à Recouvrement

Lorsque vous boulonnez deux pièces de métal ensemble, vous créez un espace microscopique entre elles.
*   **Action Capillaire** : L'eau, les liquides de nettoyage et le produit seront aspirés dans cet espace.
*   **Corrosion** : Ce fluide piégé devient stagnant, entraînant une corrosion caverneuse.
*   **Croissance** : Les bactéries prospèrent dans ces espaces humides et sombres.

## La Soudure Sanitaire

Le mobilier de salle blanche doit être **Entièrement Soudé**.
*   **Scellé** : Le joint est fusionné par soudage TIG.
*   **Meulé** : Le cordon de soudure est meulé à fleur.
*   **Poli** : La zone est polie pour correspondre au métal parent.

## Conclusion

Les boulons se desserrent. Les soudures sont éternelles. Dans un environnement critique, la permanence équivaut effectivement à la sécurité.
