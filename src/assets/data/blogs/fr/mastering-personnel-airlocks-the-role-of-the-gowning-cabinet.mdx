## Le Vêtement est le Filtre

Dans une salle blanche, l'humain est la chose la plus sale. Le filtre de la combinaison Tyvek est la seule chose qui protège le produit.
Si cette combinaison est stockée sur une étagère en fil métallique collectant la poussière avant même d'être portée, la protection est compromise.

## Stockage à Flux Laminaire Vertical

Pour les classifications de haut niveau (Classe B), les vêtements doivent être stockés dans des **armoires filtrées HEPA**.
*   **Flux d'Air Actif** : Une unité de filtrage par ventilateur pousse l'air pur vers le bas sur les vêtements suspendus, emportant toutes les particules.
*   **Suspension** : Les vêtements doivent pendre librement, sans se toucher ni toucher le sol.

## Conclusion

Traitez la combinaison comme un dispositif médical. Stockez-la stérile.
